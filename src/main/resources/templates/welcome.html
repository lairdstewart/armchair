About

Armchair is Beli for books. You get two lists: fiction and non-fiction. As you add books you'll rank them head-to-head. Like Beli, this provides a better signal of true book preferences and enables better recommendations.

Privacy

Anyone can create lists and get recommendations. To save your lists, you'll need to sign up. User authentication is through Google. Armchair is not given your email, profile picture, or account data. Lists are private by default.

Support

The best way to support is by creating an account and ranking books. This improves the recommendation algorithm for everyone. Report bugs or send suggestions to armchair@lairdstewart.com.
